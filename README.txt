To use:
checkout to a folder
require_once c2ephp.php

If you downloaded this from trunk SVN, the tests come with no instructions.

If you downloaded this in a zipped file or from the 'release' tag then the following applies:
The examples are CLI-only.
Arguments:
pray.php takes a PRAY file (.agent,.agents,.creature,.family) file as its only argument
pray-glst.php takes a .creature or .family file as its only argument.
pray-extract.php takes a PRAY file as defined above as its only argument
